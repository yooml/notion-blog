Volante : Volante Developer's Guide : Blobs (large objects)

A blob is used to store large binary objects (e.g. an image). To create a blob call IBlob IDatabase.CreateBlob() method.

IBlob interface contains only one method System.IO.Stream GetStream() which returns a standard .NET Stream object. You can write to a stream and read from it.
